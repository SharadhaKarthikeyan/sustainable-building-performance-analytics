# Raw Data Directory

This directory contains instructions for obtaining the public benchmarking dataset used in this project.

## Suggested Dataset: Seattle Building Energy Benchmarking

To build a hybrid dataset approach, you can download the public Seattle Building Energy Benchmarking dataset.

### Steps to Download:
1. Go to the [Seattle Open Data Portal](https://data.seattle.gov/).
2. Search for **"2020 Building Energy Benchmarking"** or **"2021 Building Energy Benchmarking"** (or use the latest available year).
3. Export the dataset as a **CSV** file.
4. Save the file directly to this folder as:
   `data/raw/building_energy_benchmarking.csv`

### Role of the Public Dataset:
This public dataset is used to cross-reference real-world building baseline distributions, including:
- Building types (e.g., Office, Retail, Residential)
- Floor area distributions
- Energy Use Intensity (EUI) benchmarks
- Greenhouse Gas (GHG) emissions benchmarks

### Fail-safe execution:
If you do not place the real dataset in this directory, the Python loading and cleaning scripts will automatically fall back to using the synthetic building operations and design data generated by our custom data generator, allowing the entire pipeline to run seamlessly without errors.
